Here is an example of a command

/f TaleOfTwoCities.txt /min 1 /max 100000 /d /threads 7


Command to operate on a certain file
/f TaleOfTwoCities.txt 

Command to enter min pattern length
/min 1 

Command to enter max pattern length, this can be as big or as little as the file size
/max 100000 

Command to display number of threads found per level
/d 

Command to use number of threads
/threads 7
